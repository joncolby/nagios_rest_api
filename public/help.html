nagios rest api commands

Notes:

Examples:

get a list of all hosts in Nagios (takes ~5 seconds) 

curl -o -  https://muninmx.unbelievable-machine.net:9090/hosts

get detailed status information about host and service running on 'myhostname' (takes ~20 seconds) 

curl -o -  https://muninmx.unbelievable-machine.net:9090/hosts/myhostname

find all hosts matching pattern *ghei* (make sure you put the url in 'single quotes') 

curl -o -  'https://muninmx.unbelievable-machine.net:9090/hosts/find/*ghei*'

set 10 minutes of downtime for service 'swap_utilisation' on host 'myhostname' 

curl -o -  -X PUT -d service=swap_utilisation -d duration=10 https://muninmx.unbelievable-machine.net:9090/hosts/myhostname/downtime

set 60 minutes of downtime for host 'myhostname' 

curl -o -  -X PUT -d duration=60 https://muninmx.unbelievable-machine.net:9090/hosts/myhostname/downtime

remove downtime for host 'myhostname' 

curl -o -  -X PUT  https://muninmx.unbelievable-machine.net:9090/hosts/myhostname/nodowntime

acknowledge 'disk_utilization' service problem 'myhostname' 

curl -o -  -X PUT -d service=disk_utilisation https://muninmx.unbelievable-machine.net:9090/hosts/myhostname/ack

remove acknowledgement for 'disk_utilization' service problem 'myhostname' 

curl -o -  -X PUT -d service=disk_utilisation  https://muninmx.unbelievable-machine.net:9090/hosts/myhostname/unack

disable notifications for host 'myhostname' 

curl -o -  -X PUT https://muninmx.unbelievable-machine.net:9090/hosts/myhostname/disable
